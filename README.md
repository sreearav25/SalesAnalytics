SalesAnalytics ‚Äî Simple Overview



SalesAnalytics is an end-to-end project that helps analyse:



\- Sales  

\- Profit  

\- Salary expenses  

\- Tax impact  

\- Employee costs  

\- Revenue prediction using machine learning  



The project includes three interfaces:



\### \*\*1. CLI App (Terminal Program)\*\*  

Used for technical users who prefer typing commands.  

Runs KPIs, prints summaries, runs models.



\### \*\*2. Streamlit Dashboard (Web App)\*\*  

A user-friendly interface with:



\- Sliders for salary, tax, and expenses  

\- Charts for profit, revenue, salary trends  

\- Employee add/update/delete  

\- ML prediction visualisation  



No coding required ‚Äî everything runs in the browser.



\### \*\*3. Jupyter Notebook (Analysis)\*\*  

Used to explore data, check charts, and experiment with the ML model.

\## üß± How the System Works



The system is designed so that:



\- \*\*All three interfaces\*\* (CLI, Streamlit, Notebook)

\- All data is stored in a \*\*SQLite database\*\* and \*\*CSV files\*\*  

\- This makes the system reusable and maintainable
